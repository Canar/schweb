## Abstract

An exploration in static web generation using Scheme.

## Platforms 

- Chicken Scheme
- Chibi Scheme
- Guile Scheme

## Usage

Doesn't even do anything yet except run some tests.

`./schweb-[platform] --test` runs tests on a platform.

`./test` runs tests across all platforms.
